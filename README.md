# vue-ueberstunden-app

## Project setup
```
npm install
```

Be sure to activate an environment variable named VUE_APP_API_KEY holding the Toggl API key.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
